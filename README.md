# juggernaut
Thirteenth weekly exercise.

##### CLI Commands

``` bash
# serve prod. full stack app @ port 1234
npm run start

# install dependencies server
npm run preinstall

# build server @ server/dist directory
npm run build:server

# run server @ port 1234 on debug
npm run dev:server

# install dependencies client
npm run install

# build client @ client/build directory
npm run build

# run client @ port 3000 on debug
npm run dev:client

# linting (server & client)
$ npm run lint:server
$ npm run lint:client

# code formatting (server & client)
$ npm run format:server
$ npm run format:client

```
